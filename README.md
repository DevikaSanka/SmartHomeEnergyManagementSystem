🏠 Smart Home Energy Management System

A web-based application designed to monitor, analyze, and optimize energy consumption in a smart home environment. This system helps users track device-level power usage, visualize energy data, and make informed decisions to reduce electricity consumption.

📌 Features
⚡ Real-time energy consumption tracking
📊 Interactive dashboard with graphs and statistics
🔌 Device-wise power monitoring (Light, Fan, AC, TV, etc.)
🧠 Energy usage analysis and insights
📉 Daily/Monthly consumption visualization
👤 User authentication (Login/Register)
📱 Responsive and user-friendly interface
🛠️ Tech Stack

Frontend:
React.js
CSS

Backend:
Node.js
Express.js

Database:
MongoDB Atlas

🚀 How It Works
User logs into the system
Dashboard displays energy usage data
User adds or manages smart devices
System calculates energy consumption based on device power ratings
Data is visualized through charts and graphs
User analyzes usage and optimizes energy consumption
